Solr schema design (30 min) 
===========================

In this lab we will design a meaningful schema for tweet documents. 
This lab builds on the 'indexing' lab

Lab Goals:

1. Design schema for tweets
2. Import and search the results

== STEP 1) Design the schema

  Start from the 'example' schema
  Modify it to include the tweet documents fields

== STEP 2) Showcase your schema 
  
  Hint: look in the schema browser
