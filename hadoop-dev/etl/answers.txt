Cleanup
------
    billing_cleaned =  FILTER billing_raw BY cost > 0;


Join
---
customers = LOAD '<your name>/billing/customers' USING PigStorage(',') AS (custid2:int, name:chararray, email:chararray,  state:chararray, discount:int);
joined = JOIN billing_cleaned BY custid,  customers BY custid2;

Calculate Invoices:
-----
hive query for invoice:
    SELECT custid, SUM(cost) AS total from <your name>_customer_billing GROUP BY custid;

storing invoice results
    INSERT OVERWRITE TABLE <your name>_invoices SELECT custid, SUM(cost) AS total from <your name>_customer_billing GROUP BY custid;
